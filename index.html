O qué base64?

A Base64 é uma base numérica o qual contém 64 algoritmos de 0 a 63, contendo nele todas as letras do alfabeto, tanto maiúsculas (A-Z) quanto as minúsculas (a-z), os números de 0 – 9, e por último temos os dois caracteres especiais (\ e +) para completar a tabela com 64 algoritmos.

Ela é um algorítimo para codificação de informações para transferência na internet. Geralmente é utilizada quando há necessidade de transmitir dados binários (um sistema numérico) para um dispositivo ao qual trabalha com dados textuais.

O que é Cifra de César?

A Cifra de César é uma técnica de criptografia bem simples muito conhecida. Ela é uma cifra de substituição (um sistema pré-definido de substituição.) no qual cada letra de um determinado texto será substituída por outra do alfabeto, deslocando um certo número de posições de forma constante, ou seja, sempre será utilizada as mesmas letras. Por exemplo, iremos utilizar uma troca de três posições, A iria ser substituído por D, B por E… H por K e assim com todas as letras do texto a ser criptografado.
